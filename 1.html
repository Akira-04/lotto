Starting WebPrograming: TAG, the basic

Tags of HTML

Not a big deal. TAG is a basic syntax of the HTML code. Simple is the important. With the sign <>, you can make a lot of "tricks" of the HTML code.

Basically, the parent tags are these:
!DOCTYPE html
html
head
body
These point out that this script is made up of the syntax of HTML, or this is the head part, body part, or etc.

Tag "title" sets the title of the webpage.

Tag "h1" to "h6", headlines.
The smaller the number is, the bigger the words become.

Tag "strong" makes the word to bolded one.

Tag "u" underlines the word.

Tag "p" makes the paragraph, and using CSS, as "p style="margin-top:40px"", you can specify it.
Also there is the tag "br". This tag is the enter key of the html world. It just makes the new line.

Tag "img" paste images.
Attribution "scr" is the abbreviaition of 'source'; this insists a name of the image file you want to show up.
Plus, attribution "width" litteraly sets the width of the image.
you can use this attribution as 'width="100%"'

Tag "a", which means 'anchor', links a webpage to the other.
Basically, attribution "href" is essential. It is the domain of the webpage you want to link toward.
